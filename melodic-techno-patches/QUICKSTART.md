# Melodic Techno - Quick Start

## Launch

```bash
./run_looper.sh melodic-techno-patches
```

## The 3 Patches

**q = Patch A** - Hypnotic, minimal (opening/reset)  
**w = Patch B** - Evolving, arpeggiated (main groove)  
**e = Patch C** - Peak, intense (maximum energy)

## Essential Moves

### Starting a Set
1. Load Patch A (q)
2. Start with filter closed (only bass)
3. Gradually open filter over 2-3 loops
4. Melody reveals slowly = hypnotic entry

### Building Energy
1. Switch to Patch B (w)
2. Full filter open
3. Play with tom/clap layers (mute/unmute)
4. Ride for 3-5 minutes

### Peak Moment
1. Switch to Patch C (e)
2. Everything firing at full intensity
3. Wide filter sweeps
4. Stay for climax, then...

### Creating Contrast
1. Jump back to Patch A (q)
2. Close filter immediately
3. Dramatic drop = huge impact
4. Rebuild slowly
